# 🌐 templates-wan-connectivity.nac.yaml File

The **templates-wan-connectivity.nac.yaml** file in the `data/` folder defines the following template:

 - Connectivivity monitoring configuration - defined in the **app_connectivity_monitoring** template

The template defines the following variables:

- None

Uncomment the configuration within the template to enable connectivity monitoring configuration and add the **appl_comonitoring** template to your **pods_variables.nac.yaml** file to apply to desired network.
